# 🛠️ Family Dev

Collaborative development projects managed by Smart-Buddy.

## How it works
1. Send `/dev <your idea>` in the WhatsApp group
2. Smart-Buddy brainstorms, designs, plans, and builds it
3. Results are deployed on Netlify automatically

## Tech Stack
- React / Next.js / Expo (depending on project)
- Deployed on [Netlify](https://app.netlify.com)
- Managed via GitHub (GabrielBertoli)
